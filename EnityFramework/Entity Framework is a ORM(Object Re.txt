Entity Framework is a ORM(Object Relational Mapping)

convert Database object classes and methods

Employee class can be converted into a table
A table can be converted into  a class

1.DataBase First Approach:
	Entity frame work on Data base, table can be converted into class.
	In DataBase First Approach Database is the priority.
	Database and Tables can be converted into classes and methods.
2.Code First Approach


In this approach we create classes and methods , these will
be converted into  Database and tables.

        }

            tbl_employee employee = db.tbl_employee.Find(empId);
            return employee;

            return db.SaveChanges()==1;
